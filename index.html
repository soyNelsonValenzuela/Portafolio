Juego Secreto

Es un juego desarrollado durante la etapa de formación en lógica de programación del programa Oracle ONE impartido por Alura Latam. En el cual el usuario tiene que adivinar un número secreto entre 1 y 10. Cada que el usuario ingrese una respuesta errada el sistema indicara si el número secreto es mayor o menor. Y cuando el usuario acierte se le indica en cuantos intentos lo logró.
Se utilizo Java Script para programar la lógica de este ejercicio.
